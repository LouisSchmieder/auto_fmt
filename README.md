## A auto format tool for V projects.
Auto fmt formats all changed V files which was made.

## Installation
```bash
git clone https://github.com/LouisSchmieder/auto_fmt.git
cd auto_fmt
v .
chmod +x auto_fmt
sudo mv auto_fmt /usr/local/bin
```

## Usage
```bash
auto_fmt
```
